## UC7 – Remove Food From Daıly Intake List
**Actor:** User  

**Preconditions:** User is on the first page of the navigation bar after launching the application. 

**Main Success Scenario:** 
--- 

1. User navigates to third page of navigation bar (page should shown todays intake list).

2. (Optional) User taps the **calendar icon** at the top of right the screen.

    - Application shows a calendar.

    - User selects a date from the calendar.

    - Application displays selected date's daily intake list. İf the list is empty, "this list is empty" message is shown.

3. User taps the **remove icon** to the right of the food they want to remove.

4. A confirmation pop-up is shown.

5. User taps the **confirm** button.

6. The selected food is removed from the daily intake list.

**Alternative Scenario:**
---

5a. User taps the **cancel** button.

    - The food is not removed from the daily intake list.
  
<br>

**Result:** Food is removed from the daily intake list
