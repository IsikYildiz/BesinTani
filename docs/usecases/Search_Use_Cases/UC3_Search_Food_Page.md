## UC3 â€“ Search Food's Page Manually

**Actor:** User  

**Preconditions:** User is on the first page of the navigation bar after launching the application.  

**Main Success Scenario:** 
--- 

1. User navigates to second page of navigation bar.

2. User taps the **search field** at the top of the screen.

3. User writes the food they want to search for.

4. Application displays matching food items in a dropdown list beneath the search field.

5. User taps on the food they want to see.

6. Application displays corresponding food's information.

**Alternative Scenario:**
---

4a. Instead of using the dropdown, the user taps the **search icon** to the right of the search field. 

  - The application displays the search results across the entire second page.  

  - User taps on the food they want to see.

  - Application displays corresponding food's information page.
  
<br>

**Result:** Searched food's information will be shown.
